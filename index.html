Makeshift Ring Camera 2024
Embedded Systems · ESP8266
C++PythonESP8266HC-SR04FlaskOpenCVPlatformIO

DIY security camera system using an ESP8266 microcomputer, HC-SR04 ultrasonic sensor. and logitech webcam. The sensor polls distance every 10 microseconds. When proximity drops below 50cm, it triggers a Flask server that opens a webcam feed. OpenCV scans each frame for a face with an 8 frame confidence threshold, then fires an email alert via a custom notification script.
